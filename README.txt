# Nombre del Proyecto

Este proyecto implementa un algoritmo de aprendizaje por refuerzo utilizando una Red Neuronal Profunda (DQN). El objetivo es maximizar la recompensa en un entorno específico.

## Comenzando

Estas instrucciones te permitirán obtener una copia del proyecto en funcionamiento en tu máquina local para propósitos de desarrollo y pruebas.

### Prerrequisitos

Necesitas tener instalado Python y PyTorch en tu máquina. Puedes instalarlos con los siguientes comandos:

```bash
pip install python
pip install torch
```bash

### Instalación
```bash
git clone https://github.com/tu_usuario/tu_repositorio.git
cd tu_repositorio
```bash

### Contribuyendo
Todavia queda subir la parte del arduino uno!
